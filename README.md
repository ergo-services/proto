## erlang23
[![BSL 1.1](https://img.shields.io/badge/license-BSL_1.1-31af90.svg)](erlang/LICENSE)

The Erlang Network Stack (EPMD, ETF and DIST) implementation for the Ergo Framework

Supports OTP-23...OTP-27
